# Ordena-o-analise-de-desempenho-C
Este programa em C é uma ferramenta de análise de desempenho que implementa e compara três algoritmos clássicos de ordenação: Bubble Sort, Insertion Sort e Heap Sort. O foco é medir o número de passos fundamentais (comparações e trocas/movimentações) e o tempo de execução real em diferentes tamanhos de input ($N$).
